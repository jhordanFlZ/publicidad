# Guion: El Condicional IF - Estilo Pixar 3D
# Fecha de creacion: 2026-02-18
# Duracion: 75-90 segundos
# Plataformas: TikTok / Reels / YouTube Shorts

---

## TEMA DEL VIDEO
El condicional IF en programacion explicado como una pelicula de animacion Pixar 3D

## PUBLICO OBJETIVO
Personas sin experiencia en programacion (de 12 a 35 anos), audiencia general de redes sociales que nunca ha escrito codigo pero usa tecnologia todos los dias.

## CONCEPTO CREATIVO
Un guardia de seguridad llamado IF trabaja en la puerta de un club exclusivo llamado "El Codigo". Cada noche debe tomar la decision mas importante del universo: dejarte pasar o no. La metafora del portero es universal, visual e inmediatamente graciosa.

## GUION DETALLADO

### [0:00-0:03] HOOK

DESCRIPCION VISUAL: Primer plano cinematografico de una puerta enorme y dorada con el numero "01" grabado en relieve. Luz de neon azul y morado. Musica de fanfarria que se corta en seco. De la oscuridad emerge un personaje enorme, cuadrado como un bloque de Lego pero con cara expresiva estilo Pixar: cejas gruesas, ojos grandes y brillantes, bigote militar. Lleva una gorra de guardia de seguridad con el simbolo ">".

DIALOGO (voz grave, seria, con acento dramatico):
"Yo... soy IF."

TEXTO EN PANTALLA (letras enormes que caen como bloques):
"IF: El guardia mas poderoso del universo"

ENERGIA: Maximo dramatismo. Silencio antes de la frase. Como el opening de una pelicula de accion.

---

### [0:03-0:10] PROBLEMA

DESCRIPCION VISUAL: Camara se aleja. Vemos que IF no esta en una pelicula de accion. Esta parado frente a la puerta de un club de barrio mediocre con un letrero de neon roto que dice "EL CODIGO". Hay una sola persona en la fila: un personaje pequeno y nervioso llamado USUARIO, con sudadera con capucha y zapatillas viejas. El contraste es comico.

NARRACION EN OFF (voz caida, desinfladada):
"Cada noche, IF tiene que tomar la decision mas importante del universo..."

IF se inclina hacia abajo, inspecciona a USUARIO con una lupa gigante.

USUARIO (voz temblando):
"Ey... hola. Vengo a... programar?"

IF levanta una ceja. Pausa larga. Musica de suspenso ridiculamente exagerada.

---

### [0:10-0:40] EXPLICACION

DESCRIPCION VISUAL: La escena se transforma. El mundo se vuelve 2D animado como una pizarra magica. IF explica caminando por el "mundo interior" donde todo es logica pura: arboles en forma de llaves de codigo, nubes que forman parentesis.

IF saca un cuaderno enorme con una sola pagina. En ella dice:

  SI (condicion es verdadera)
      -> HAZ ALGO
  SINO
      -> HAZ OTRA COSA

IF (didactico, con pasion de maestro que ama demasiado su trabajo):
"Mira, es simple. Yo solo hago UNA pregunta. Una. Sola. Pregunta."

Aparece una pantalla holografica gigante con el codigo:

  if (edad >= 18):
      print("Puedes pasar")
  else:
      print("Vuelve en unos anos")

IF senala cada parte como si dirigiera una orquesta:

"Si la respuesta es VERDADERO..."
(flecha verde gigante apunta a la izquierda, se abre una puerta dorada con confeti)
"...entras."

"Si la respuesta es FALSO..."
(flecha roja apunta a la derecha, aparece una pared de ladrillos)
"...no entras."

NARRACION EN OFF:
"El IF es como un semaforo. Verde o rojo. Si o no. Verdadero o falso. No existe el amarillo."

IF se gira a camara, orgulloso.

IF:
"Binario, bebe."

---

### [0:40-0:60] SKETCH / ESCENA COMICA

DESCRIPCION VISUAL: Regresamos a la puerta del club. Tres personajes nuevos se acercan a la fila. IF los evalua uno por uno como un juez de "Got Talent" pero de logica.

PERSONAJE 1: Un numero enorme y brillante, el "18".
IF revisa su lista.
IF: "IF edad == 18... VERDADERO."
Puerta se abre. Musica de victoria. El 18 entra bailando.

PERSONAJE 2: Un numero tembloroso, el "15". Cara de esperanza.
IF lo mira. Saca una calculadora. El resultado dice "FALSO" en rojo.
IF senala la salida sin decir nada, con cara de piedra.
El 15 se va llorando. Una lagrimita en camara lenta. Musica de violines tragicos.

PERSONAJE 3: Una variable llamada "X" con lentes de intelectual y corbata.
IF:
"A ver... IF x > 10... Hmm. Pero... que ERES tu, X?"

X (nervioso, ajustando los lentes):
"Soy... indefinido?"

IF explota de risa. Se dobla. Se cae al piso.

IF (entre carcajadas):
"ERROR. ERROR. NullPointerException. Ay, X..."

TEXTO EN PANTALLA: "Siempre valida tus variables, amigos."

---

### [0:60-0:75] EJEMPLO PRACTICO

DESCRIPCION VISUAL: IF se limpia las lagrimas. La camara se acerca. Modo serio pero amigable. Detras de el aparecen ejemplos visuales flotantes como burbujas.

IF:
"Oye. El IF esta en TODOS lados."

Las burbujas explotan revelando escenas cotidianas:
- Netflix en una pantalla: "IF tienes suscripcion → reproduce / ELSE → pantalla de pago"
- Un semaforo: "IF luz es roja → para / ELSE → avanza"
- Un telefono con notificacion: "IF mensaje nuevo → suena / ELSE → silencio"

IF (con voz de revelacion):
"Tu vida... ya corre en IFs. Solo que ahora sabes como se llama."

Pausa. IF guina un ojo a camara.

---

### [0:75-0:90] CTA

DESCRIPCION VISUAL: IF vuelve a su puerta. USUARIO sigue en la fila. IF lo mira. Consulta el cuaderno. Asiente muy lentamente con cara de juez severo.

IF (voz grave y oficial):
"IF... eres mi seguidor..."

Pausa larga. Tension musical.

IF (explota en una sonrisa enorme, el personaje mas expresivo del mundo):
"...ENTONCES... dale like y comenta tu primer IF."

La puerta dorada se abre de par en par. USUARIO entra con cara de incredulidad y alegria. Confeti digital explota. Musica de celebracion.

TEXTO EN PANTALLA (con animacion de letras que rebotan):
"Escribe en comentarios: if (yo_codifico): print('true')"

IF mira a camara, levanta el pulgar.

IF:
"Bienvenido al codigo."

FADE TO BLACK.

LOGO al final: llaves de codigo { } con cara de Pixar. Ojos grandes. Guino.

## ELEMENTO HUMORISTICO PRINCIPAL

El contraste entre la seriedad exagerada de IF (que se comporta como personaje de pelicula epica) y lo mundano de su trabajo real (portero de un club mediocre) es el motor comico central. El gag del "X indefinido" es el pico del humor porque conecta con un error real de programacion (variable no definida / NullPointerException) de manera completamente visual y sin requerir conocimiento previo. La audiencia tecnica rie por reconocimiento; la audiencia general rie porque IF se cae al piso de la risa.

## ESTILO VISUAL

- Paleta de colores: azul marino profundo + morado neon + amarillo electrico + blanco puro para el codigo
- Tipografia: monospace para el codigo (estilo terminal), display redondeada para titulos estilo Pixar
- Transiciones: cuts rapidos en la seccion comica, movimientos de camara lentos y cinematograficos en el hook
- Texturas: los personajes tienen piel con subsurface scattering tipico de Pixar (ligeramente traslucida)
- Iluminacion: rim light en IF para darle dimension de heroe epico; luz de neon sucia en el exterior del club para el contraste comico
- Graficos de codigo: holograficos, con efecto de escritura en tiempo real, particulas de luz azul flotando

## NOTAS DE PRODUCCION

Para el animador:
IF debe tener proporciones ligeramente exageradas: manos enormes, cuerpo cuadrado pero con movimientos fluidos tipo animacion de alto presupuesto. Sus expresiones deben ir del 0 al 100 rapidamente, caracteristica clave del acting de Pixar.

Para el editor:
El corte entre la fanfarria epica del hook y la revelacion del club mediocre es el primer chiste. El timing es todo: si el corte tarda un frame de mas, el gag se pierde. Practicar este cut hasta que sea instantaneo.

Duracion objetivo por plataforma:
- TikTok y Reels: 75 segundos (cortar el ejemplo practico a 5 segundos)
- YouTube Shorts: 90 segundos completos
- Version larga opcional: 3 minutos agregando ELSE IF como segundo personaje (hermano menor de IF, mas nervioso, siempre dice "pero... y si...?")

Accesibilidad:
Todo el codigo que aparece en pantalla debe tener subtitulos en espanol explicando cada simbolo. El video debe funcionar en mute (texto en pantalla) y con audio (narracion completa). Esto maximiza el alcance organico en redes donde el autoplay es en silencio.
